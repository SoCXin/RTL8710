## [简介](https://github.com/mcuyun/RTL8710/wiki) 

瑞昱RTL8710是一个完整且自成体系的 WiFi 网络解决方案，集成了一个 ARM Cortex M3 MCU、802.11n 无线网络控制器等于一体，能够独立运行，也可以作为从机搭载于其他主机 MCU运行。
